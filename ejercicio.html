Selectores

Selector de Elementos

El selector de elementos se utiliza como su nombre lo dice para seleccionar elementos dento del HTML.
en el cual se puede definir por ejemplo la fuente, el tamaño y color de texto.


Selector de Identificador

El selector de id utiliza el atributo id de un elemento HTML para seleccionar un elemento específico.
El id de un elemento es único dentro de una página, ¡así que el selector de id se utiliza para seleccionar
un elemento único!.


Selector de Clase

El selector de clases selecciona elementos HTML con un atributo de clase específico.
Para seleccionar elementos con una clase específica, escriba un carácter de punto (.), seguido del
nombre de la clase.
